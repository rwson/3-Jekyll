vue-router分析

Jul 25, 2017

随着前端应用的业务功能越来越复杂、用户对于使用体验的要求越来越高，单页应用（SPA）成为前端应用的主流形式。在单页应用中, 路由几乎是必不可少的一部分。

我们都知道, 在浏览器端, 可以用hash和history来实现路由, 前者是在url后面加#/xxx的方式来实现, 后一个则是/xxx的方式来实现, 需要注意的是, 后者是在HTML5里面新增的方法, 所以, 如果考虑到兼容浏览器的情况, 还是选择前者比较适合(不过现在很少人用低版本IE了😂)

vue作为一个被很多开发者亲睐的前端框架, 配合最多的路由系统就是vue-router。在vue-router里面, 是通过配置项中的mode来执行具体选择哪种模式的, 下面我们一起看个简单的配置:
